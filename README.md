# 📊 Simulador de Distribuciones de Probabilidad

Este proyecto es una herramienta computacional interactiva desarrollada en Python para simular, analizar y visualizar distribuciones de probabilidad discretas y continuas.

Desarrollado para la asignatura de **Probabilidad y Estadística** en **ESCOM - IPN**.

---

## 🛠️ Requisitos e Instalación

Antes de ejecutar el simulador, instale las librerías necesarias ejecutando el siguiente comando en la terminal:

```bash
pip install streamlit numpy scipy matplotlib seaborn pandas
```

---

## 💻 Cómo ejecutar el programa

Una vez instaladas las dependencias, inicie la aplicación con el siguiente comando:

```bash
streamlit run app.py
```

Después de presionar **Enter**, se abrirá automáticamente una pestaña en el navegador web con la interfaz gráfica interactiva lista para utilizarse.
